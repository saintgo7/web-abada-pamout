# User Guide
**Program Portfolio Management System**

## 1. Getting Started
1. Visit https://ppms.abada.com
2. Enter email/password
3. Click [Login]
4. (Optional) MFA code

## 2. Key Features
- **Create Program**: [Programs] → [+ New Program]
- **Allocate Resources**: [Resources] → Drag team member to project
- **Generate Report**: [Reports] → Select template → [Generate]

## 3. Tips
- **Shortcuts**: Ctrl+K (search), Ctrl+N (new)
- **Personalize**: Reorder dashboard widgets
- **Favorites**: Star frequently used menus

## 4. FAQ
**Q: Forgot password?**
A: Click [Forgot Password] on login page

**Q: Export data?**
A: Yes, use [Export] button on any page

**Q: Multi-language support?**
A: Yes, Korean, English, Japanese
