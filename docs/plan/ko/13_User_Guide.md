# 사용자 가이드
**프로그램 포트폴리오 관리 시스템**

## 1. 시작하기

### 1.1 로그인
1. https://ppms.abada.com 접속
2. 이메일/비밀번호 입력
3. [로그인] 클릭
4. (선택) MFA 코드 입력

### 1.2 대시보드 투어
- 왼쪽: 네비게이션
- 중앙: 메인 콘텐츠
- 오른쪽: 알림, 프로필

## 2. 주요 기능

### 2.1 프로그램 생성
1. [프로그램] 메뉴 클릭
2. [+ 새 프로그램] 클릭
3. 이름, 설명, 기간 입력
4. [저장] 클릭

### 2.2 리소스 할당
1. [리소스] 메뉴 클릭
2. 리소스 로드맵 뷰 선택
3. 팀원을 프로젝트로 드래그
4. 할당 비율 설정
5. [저장] 클릭

### 2.3 보고서 생성
1. [보고] 메뉴 클릭
2. 템플릿 선택 또는 커스텀
3. 기간, 필터 설정
4. [생성] 클릭
5. PDF/Excel로 내보내기

## 3. 팁

### 3.1 단축키
- Ctrl + K: 검색
- Ctrl + N: 새 항목
- Ctrl + /: 단축키 도움말

### 3.2 개인화
- 대시보드 위젯 순서 변경
- 자주 사용는 메뉴 즐겨찾기
- 알림 설정

## 4. FAQ

**Q: 비밀번호를 잊어버렸어요**
A: 로그인 페이지에서 [비밀번호 찾기] 클릭

**Q: 데이터를 내보낼 수 있나요?**
A: 네, 모든 페이지에서 [내보내기] 가능

**Q: 여러 언어를 지원하나요?**
A: 네, 한국어, 영어, 일본어 지원
